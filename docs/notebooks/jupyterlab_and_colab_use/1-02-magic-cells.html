Uso de %%bash y %%writefile

  • 5:00 min | Última modificación: Septiembre 13, 2021.

Los ambientes tienen varias utilidades que extienden las capacidades de los notebooks más allá de las características del kernel que se este usando. Las magics permiten modiicar el comportamiento de las celdas individuales. A continuación se describen las principales. Haga clic aquí para acceder a la documentación oficial.

Una lista de los magics disponibles se encuentra en: https://github.com/ipython/ipython/wiki/Extensions-Index

## %%bash

Ejecuta el código de la celda usando bash.

%%bash
ls -1 sample_data/*

!ls -1 sample_data/*

files = !ls -1 sample_data/*
files

%%writefile

Permite grabar el contenido de la celda al archivo especificado.
